# Ghost Blog

Ghost est une plateforme de publication open source puissante et flexible, conçue pour les blogueurs et les éditeurs professionnels. Ce template Docker vous permet de déployer facilement une instance de Ghost avec une base de données MySQL.

## Fonctionnalités

*   **Newsletter intégré** : Gérez et envoyez des newsletters directement depuis votre tableau de bord.
*   **Membres & abonnements** : Offrez du contenu exclusif et gérez les abonnés payants.
*   **API headless** : Utilisez Ghost comme un CMS headless pour construire des expériences personnalisées.
*   **Éditeur moderne** : Un éditeur de contenu intuitif et riche en fonctionnalités.
*   **Thèmes personnalisables** : Adaptez l'apparence de votre blog avec des thèmes flexibles.

## Déploiement

Ce template utilise Docker Compose pour orchestrer deux services :

*   **`ghost`** : L'application Ghost elle-même, basée sur l'image `ghost:5.101-alpine`.
*   **`db`** : Une base de données MySQL, basée sur l'image `mysql:8.4`.

Les données de Ghost (contenu, images, thèmes) et de la base de données sont persistées via des volumes Docker nommés (`ghost_content` et `ghost_db`).
